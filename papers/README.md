# CADE Research Papers & References

This directory collects research papers, academic references, and technical reports relevant to the CADE project.

## Topics

- **Attack graph construction and analysis** — foundational graph-based vulnerability analysis
- **Cyber kill chain and ATT&CK framework** — threat actor modeling and TTP taxonomy
- **Risk scoring methodologies** — CVSS and beyond; temporal and environmental scoring
- **Explainable AI for security** — making detection outputs interpretable for analysts
- **Anomaly detection in network traffic** — statistical and ML-based approaches
- **Federated learning for threat intelligence** — privacy-preserving collaborative defense

## Adding Papers

When adding a paper or reference:
1. Place the PDF (if redistributable) or a `.bib` / `.md` citation file in a subdirectory named after the topic area.
2. Update this README with a brief entry in the relevant section below.
3. Note the license or access restrictions for any included PDFs.
